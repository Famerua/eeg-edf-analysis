# EDF Analysis Notes

В проекте используются 6 каналов EEG:
`O1`, `T3`, `Fp1`, `Fp2`, `T4`, `O2`.

## Какие файлы смотреть

- [main.ipynb](/Users/alexandr/taubay/main.ipynb): чтение одного EDF-файла и базовая проверка структуры записи.
- [filter.ipynb](/Users/alexandr/taubay/filter.ipynb): очистка всех `1-9` EDF-файлов по эпохам по 10 секунд, удаление эпох с артефактами и сохранение очищенных записей в `cleaned_data/`.
- [Define79.ipynb](/Users/alexandr/taubay/Define79.ipynb): определение, где в файлах `7` и `9` более вероятны открытые или закрытые глаза по альфа-пику.
- [compare.ipynb](/Users/alexandr/taubay/compare.ipynb): сравнение очищенных файлов `3-9` с эталонами `1` и `2`.
- [cleaned_data/cleaning_summary.csv](/Users/alexandr/taubay/cleaned_data/cleaning_summary.csv): сводка по очистке, включая вырезанные эпохи.
- [cleaned_data/cleaning_summary.json](/Users/alexandr/taubay/cleaned_data/cleaning_summary.json): те же результаты в JSON.

## Что было сделано

- Файлы `.edf` были разбиты на эпохи по 10 секунд.
- Для каждой эпохи и каждого канала считался максимум абсолютной амплитуды.
- Для каждого канала считался типичный максимум по всем эпохам через `median`.
- Эпоха помечалась как артефактная, если максимум в этой эпохе слишком сильно превышал типичный максимум канала.
- После этого хорошие эпохи склеивались обратно и сохранялись в `cleaned_data/*.edf`.
